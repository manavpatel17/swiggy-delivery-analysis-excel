Swiggy Data Analysis Dashboard 📊

Project Overview:
This project involves a comprehensive analysis of Swiggy's delivery and sales data. I utilized Excel to clean, transform, and visualize a large dataset to help business stakeholders understand performance trends across India.
(Note: If you put your image in a folder, change this to: images/image_aa06b2.png)


🛠️ Key Technical Skills Used:
Data Cleaning: Handled missing values, standardized city names, and formatted date columns using Power Query.
Data Transformation: Created calculated columns for Quarters, Days, and Weekly Trends to enable deeper time-series analysis.
Pivot Tables & Modeling: Used Pivot Tables to aggregate large amounts of row data into meaningful summaries.

Advanced Visualizations: Designed a user-friendly UI with:
 KPI Cards: For instant snapshots of Sales, Ratings, and Order Volume.
 Map Charts: To visualize geographic sales distribution.
 Interactive Slicers: Allowing users to filter by Month, State, and Food Category.


📈 Business Insights Discovered:
Peak Sales Performance: Saturday is the most profitable day of the week, reaching ₹7.78M in sales.
Customer Preferences: 65% of the order volume comes from Vegetarian food items.
Top Growth Hub: Bengaluru leads all cities with total sales of ₹5.46M.
Customer Satisfaction: Maintained a strong average rating of 4.34 across the platform.


📂 How to Explore this Project
Download the Swiggy_Data_Analysis_Project.xlsx file from this repository.
Open the file in Microsoft Excel.
Navigate to the "Dashboard" tab.
Use the Slicers on the left to interact with the charts and see how data changes in real-time.
